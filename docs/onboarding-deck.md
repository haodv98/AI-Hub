# Batch Onboarding Deck (Phase 3)

## Session Structure (30 minutes)
1. 5 min: Why AIHub (cost control + governance + zero-friction usage)
2. 10 min: Admin portal walkthrough (key retrieval, usage, policy visibility)
3. 10 min: Hands-on setup for Cursor/CLI
4. 5 min: Support path and escalation

## Team Batch Playbook
- Run onboarding in waves of 2-3 teams.
- Keep 48h hypercare window after each wave.
- Track first successful API call per user.
- Escalate issues via support mailbox and on-call IT admin.

## Setup Snippets
- Cursor: `openai.baseUrl`, `openai.apiKey`
- Claude CLI: `ANTHROPIC_BASE_URL`, `ANTHROPIC_API_KEY`
- Verification: send one test prompt and confirm usage appears in dashboard.
